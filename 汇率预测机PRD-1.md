# 汇率预测机 — 产品需求文档 (PRD v1.0)

## 1. 产品概述

### 1.1 产品定位

汇率预测机是一款面向外汇交易策略研究人员的智能分析平台，聚焦 USD/CNH（美元兑离岸人民币）汇率。平台融合实时行情数据、多因子量化算法、AI 大模型分析和新闻情报，为用户提供结构化的方向判断和策略辅助参考。

### 1.2 目标用户

- 外汇交易策略研究员
- 企业财务/资金管理人员（关注汇率风险敞口）
- 量化交易爱好者

### 1.3 核心价值

| 维度 | 价值 |
|------|------|
| 数据整合 | 多源行情数据自动采集、清洗、归一化 |
| 量化分析 | 7 大算法 + 市场状态自适应权重，输出 -100~+100 综合评分 |
| AI 解读 | 大模型结合技术面 + 消息面，输出结构化分析报告 |
| 新闻情报 | 自动抓取、去重、AI 摘要，提炼关键驱动因素 |
| 交互式对话 | 支持上下文连续对话、文件上传、历史回溯 |

## 2. 功能模块

### 2.1 行情总览 (Overview)

**用户故事**: 作为用户，我希望打开首页就能看到当前汇率走势和关键指标的概览。

**功能点**:
- 最新收盘价、涨跌幅、日内波动范围
- 迷你 K 线走势图（Sparkline）
- 最新预测结论卡片（方向 + 置信度）
- 量化信号综合评分速览
- 新闻情绪摘要

---

### 2.2 技术分析 (Analysis)

**用户故事**: 作为用户，我希望查看完整的技术指标图表和量化信号，辅助判断市场方向。

**功能点**:

#### 2.2.1 K 线图表
- 支持多周期切换：小时线 / 4 小时线 / 日线
- OHLC 蜡烛图 + 成交量柱状图
- 可叠加技术指标曲线

#### 2.2.2 技术指标面板
- 指标分组展示（趋势类、震荡类、动量类）
- 每个指标显示当前值 + 信号判定（买入/卖出/中性）
- 支持自定义指标公式（管理端配置）

#### 2.2.3 量化信号面板
- 综合评分仪表盘（-100 ~ +100，颜色编码）
- 7 轴雷达图展示各算法信号强度
- 信号卡片列表（按绝对值排序）
- 市场状态标签（上升趋势 / 下降趋势 / 震荡 / 高波动）
- 置信度百分比
- 数据源健康状态指示

---

### 2.3 AI 智能对话 (Intelligence)

**用户故事**: 作为用户，我希望通过自然语言与 AI 对话，获取专业的汇率分析和预测解读。

**功能点**:

#### 2.3.1 对话能力
- 流式输出（SSE），实时显示 AI 回复
- 支持多轮上下文对话
- 会话管理（新建 / 切换 / 删除）
- 会话历史持久化

#### 2.3.2 上下文增强 (RAG)
- 自动注入最新行情数据（近 10 日 OHLC）
- 自动注入技术指标当前值和信号判定
- 自动注入量化引擎综合评分和主要信号
- 自动注入新闻摘要和情绪分析
- 自动注入近期预测历史

#### 2.3.3 文件交互
- 支持图片上传（截图分析）
- 支持文档上传（PDF/Excel 内容提取）
- AI 结合上传内容和市场数据综合分析

#### 2.3.4 数据查询工具
- AI 可主动调用数据库查询工具
- 支持查询历史行情、预测记录、统计信息
- 最多 3 轮工具调用循环

---

### 2.4 预测历史 (History)

**用户故事**: 作为用户，我希望回顾历史预测记录，评估预测准确性和趋势变化。

**功能点**:
- 预测时间线（按时间倒序）
- 筛选：方向 / 周期 / 日期范围
- 统计面板：总数、方向分布、平均置信度
- 置信度趋势图
- 周期分布饼图
- 单条预测详情（含 rationale 和风险提示）

---

### 2.5 新闻情报

**用户故事**: 作为用户，我希望了解影响汇率的最新消息面因素。

**功能点**:
- 每日新闻摘要（AI 生成）
- 关键驱动因素提取（因素 + 方向 + 详情）
- 综合情绪判定（利多 / 利空 / 中性）
- 自动定时更新（每 30 分钟）

---

### 2.6 管理后台 (Admin)

**用户故事**: 作为管理员，我希望配置系统参数、监控运行状态、管理数据源。

**功能点**:

#### 2.6.1 系统总览
- 数据库表记录数统计
- 各定时任务运行状态
- 最近任务执行日志

#### 2.6.2 定时任务管理
- 查看所有 Cron 任务状态（运行中 / 空闲 / 错误）
- 手动触发任务（新闻抓取 / 新闻摘要 / 预测 / 量化分析）
- 中止运行中的任务

#### 2.6.3 新闻源管理
- CRUD 新闻源（名称、URL、类型、分类、抓取间隔）
- 支持类型：RSS / JSON API / 社交媒体
- 启用/禁用控制

#### 2.6.4 指标配置
- 技术指标参数编辑（周期、阈值、权重）
- 自定义公式编辑器
- 指标分组管理
- 图表类型配置（折线 / 柱状 / 子图）

#### 2.6.5 数据库浏览器
- 动态查询任意表
- 结果表格展示 + 筛选

#### 2.6.6 管理员 AI 助手
- 独立的管理员对话界面
- 可查询系统状态和数据

## 4. 定时任务策略

| 任务 | 频率 | 时间 | 说明 |
|------|------|------|------|
| 新闻抓取 | 每 5 分钟 | 全天 | 从所有启用的新闻源拉取最新文章 |
| 新闻摘要 | 每 30 分钟 | 全天 | AI 汇总近期新闻生成摘要 |
| 自动预测 | 每日 4 次 | 01:00/05:00/09:00/13:00 (工作日) | 生成 T+1/T+2/T+3 预测 |
| 量化分析 | 每日 4 次 | 02:00/06:00/10:00/14:00 (工作日) | 运行 7 算法计算综合评分 |

## 5. 非功能需求

### 5.1 性能
- AI 对话首字延迟 < 3 秒
- 页面首屏加载 < 2 秒
- 量化分析单次计算 < 10 秒

### 5.2 可用性
- 数据源多级降级（Yahoo → Frankfurter → ECB）
- 熔断机制：连续 5 次失败自动禁用 60 秒
- 任务失败自动记录日志，不影响其他任务

### 5.3 安全
- API Token 通过环境变量管理
- 文件上传限制 20MB
- 管理后台独立部署端口

---

## 6. 产品边界

### 包含
- USD/CNH 汇率分析和预测
- 技术面 + 消息面综合研判
- 量化信号计算和展示
- AI 对话式交互

### 不包含
- 实际交易执行
- 多币种同时分析
- 实时 tick 级数据
- 用户账户体系和权限管理
- 移动端适配

## 7. 术语表

| 术语 | 说明 |
|------|------|
| OHLC | Open/High/Low/Close，开高低收价格 |
| RSI | 相对强弱指标 (Relative Strength Index) |
| MACD | 移动平均收敛散度 |
| ADX | 平均方向指数，衡量趋势强度 |
| Bollinger Bands | 布林带，基于标准差的通道指标 |
| Regime | 市场状态（趋势/震荡/高波动） |
| RAG | 检索增强生成 (Retrieval-Augmented Generation) |
| SSE | Server-Sent Events，服务端推送 |
| Horizon | 预测周期（T+1/T+2/T+3 天） |
